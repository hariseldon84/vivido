# Milestone 0 — Validation Prototype

**Phase:** Pre-MLP
**Owner:** Founder solo
**Window:** Weeks 0–8
**Status:** PLANNED

---

## Purpose

Prove demand before full MLP build. This milestone is not the full Vivido product. It is the smallest believable prototype that tests whether transcript-first editing + creator-oriented export creates immediate willingness to pay.

---

## Included Scope

- transcript-first editing prototype
- creator preset / workflow framing
- lightweight Publish Package output
- minimal local telemetry
- founder-led creator validation loop

---

## Explicitly Excluded

- full GPU timeline parity
- full Audio Room
- full account/cloud/payment stack
- polished production shell
- broad platform surface area

## Exit Criteria

- 5 named validation creators are recruited
- each uses the prototype on 3 real videos
- at least 3 of 5 say “I’d pay for this today”
- feedback confirms whether GPU playback is mandatory for the MLP wedge

---

## Risk Gates

- Descript FTO review completed
- provisional patent filings sequenced per PRD
- no creator-facing demo before legal gate is cleared
